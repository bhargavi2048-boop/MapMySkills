# MapMySkills — AI Career Gap Mapper

**AI-powered skill gap analysis for 16+ career domains.**

Built by **Bhargavi N** · First Year CSE Student

---

## Quick Start

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari)
2. Select your career domain (auto-detects from JD)
3. Paste your resume OR upload a PDF
4. Paste the job description
5. Click **Analyze My Skills**
6. Click **Open Report →** for the full report

**No server required.** Runs entirely in the browser.
